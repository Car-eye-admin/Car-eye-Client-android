﻿# Car-eye-Client-android

Car-eye-Client android 版本是car-eye开源开源团队基于car-eye open API 开发的客户端软件，对接car-eye JT1077 平台。主要功能有：
1. 实时定位功能
2. 实时视频监控
3. 轨迹回放
4. 视频回放
能满足基本的车辆监管需求。
